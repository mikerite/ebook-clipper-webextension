eBook Clipper 0.9

Thank you for trying this extension.

This is an early version of the extension and the user experience is still rough. Hopefully it will still be useful to you.

How to Use

  1. Open the sidebar.
  2. Enter the ID of the element whose content you want to clip in the input labelled 'Element ID'. Typically you'll have to find out what the ID is using the developer tools. (This aspect should be improved in future versions of the extension.)
  3. Click the 'Clip' button. A new chapter should be added under the 'Chapters' heading.
  4. Optionally, enter a title for your ebook in the input labelled 'Title ID'.
  5. Click the 'Save eBook' button and download the epub file.
